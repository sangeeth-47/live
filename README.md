## Overview

ScreenShare is a lightweight browser-based screen sharing application that allows presenters to securely share their screen with multiple viewers using a short room code.

Viewers simply enter the room code, request access, and wait for presenter approval before joining the live session.

---

## Features

- Secure screen sharing
- Short room code for easy access
- Presenter approval before viewers can join
- Multiple viewers supported
- Full-screen viewing
- Low-latency streaming
- Works directly in modern web browsers
- No software installation required

---

## Getting Started

### Presenter

1. Open ScreenCode.
2. Create a new sharing session.
3. Allow browser permission to capture your screen.
4. Share the generated room code.
5. Approve viewer requests.
6. Start presenting.

### Viewer

1. Open ScreenCode with shared URL.
2. Enter the name.
3. Send a join request.
4. Wait for presenter approval.
5. Watch the live screen.

## How It Works

```text
Presenter
    │
    │ Create Session
    ▼
Generate Room Code
    │
    │ Share Code
    ▼
Viewer Joins
    │
    │ Approval Request
    ▼
Presenter Approves
    │
    ▼
Live Screen Sharing
```

## Privacy

- Screen content is shared only after explicit permission from the presenter.
- Viewers cannot join without presenter approval.
- No browser extensions are required.

---

## Requirements

- Modern Chromium-based browser (recommended)
- Stable internet connection
- JavaScript enabled
